## 화면 상세 설계

### P-010 직원 대시보드

| 항목 | 내용 |
|------|------|
| **화면 목적** | 직원이 잔여 연차·사용 이력을 확인하고, 증명서 신청으로 빠르게 이동한다. |
| **진입 조건** | 직원 계정으로 로그인 시 기본 화면 / 사이드바 [대시보드] |
| **다음 화면** | 없음 (현재 페이지 내 처리) |

**화면 구성**

| 영역 | 컴포넌트 | 상세 |
|------|---------|------|
| 잔여 연차 카드 | 잔여 일수(크게) + 총·사용 일수 + 진행 막대 | |
| 앱 설치 QR 카드 | 모바일 앱 설치 안내 | 설치 여부와 무관하게 **상시 표시** |
| 연차 사용 이력 테이블 | 날짜 / 구분 / 일수 / 사유 / 처리 상태 | **현재 연도(오늘 날짜 기준 yyyy) 이력만 표시**. 연차 사용일자 기준 **최신순** 정렬 |
| 증명서 관리 패널 | 최근 신청 내역 + [증명서 신청] 버튼 | 최근 신청 내역 최대 **5건** 표시. [증명서 신청] 클릭 시 직원 업무신청(P-011) 화면으로 이동 |

**조건부 규칙**
- 연차 신청은 모바일 MY인사 앱에서만 가능. PC 직원 화면에서는 연차 이력 조회만 지원.
- 연차 사용 이력이 없을 경우: **"내역이 없습니다."** 표시
- 증명서 신청 이력이 없을 경우: **"내역이 없습니다."** 표시
