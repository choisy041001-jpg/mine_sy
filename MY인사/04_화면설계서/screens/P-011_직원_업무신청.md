## 화면 상세 설계

### P-011 직원 업무신청

| 항목 | 내용 |
|------|------|
| **화면 목적** | 직원이 PC에서 증명서를 신청하고, 이력을 조회·관리한다. |
| **진입 조건** | 사이드바 [업무신청] |
| **다음 화면** | 없음 (현재 페이지 내 처리) |

**화면 구성**

| 영역 | 컴포넌트 | 상세 |
|------|---------|------|
| 상단 영역 | 필터 (종류·상태) + [증명서 신청] 버튼 | |
| 이력 테이블 | 신청일 / 종류 / 용도 / 처리 상태 / 액션 버튼 | 기본 정렬: **신청일 기준 최신순** / 전체 이력 표시 |
| 이력 테이블 | 대기 상태 | [신청 취소] — 클릭 시 취소 확인 팝업 표시 후 확인 시 취소 처리 |
| 이력 테이블 | 발급완료 상태 | [다운로드] — 클릭 시 즉시 증명서 파일 다운로드 |
| 이력 테이블 | 반려 상태 | [사유보기] |
| 증명서 신청 팝업 | 종류 선택 | 팝업 내 항목 선택 방식 — 재직증명서 / 경력증명서 / 기타증명서 |
| 증명서 신청 팝업 | 용도 입력 (**필수**) | 용도: 최대 100자 / 단일 라인 |
| 증명서 신청 팝업 | 기타증명서 선택 시 명칭 입력 필드 추가 | 명칭: **필수** (기타증명서일 때) / 최대 50자 / 단일 라인 |

**조건부 규칙**
- 신청 이력이 없을 경우: **"내역이 없습니다."** 표시

**빈 상태**
- 필터 조건에 해당하는 이력이 없을 경우: **"내역이 없습니다."** 표시
