## 화면 상세 설계

### M-030 증명서 관리

| 항목 | 내용 |
|------|------|
| **화면 목적** | 증명서 종류를 선택하여 신청하거나, 이전 발급 이력을 확인한다. |
| **진입 조건** | 하단 탭바 [증명서] 탭 |
| **다음 화면** | M-031 / M-032 / M-033 / M-034 / M-035 |

**화면 구성**

| 영역 | 컴포넌트 | 상세 |
|------|---------|------|
| 상단 영역 | 타이틀 | "증명서 관리" |
| 증명서 종류 카드 | 재직증명서 | 아이콘 + 설명 문구 |
| 증명서 종류 카드 | 경력증명서 | 아이콘 + 설명 문구 |
| 증명서 종류 카드 | 기타증명서 | 아이콘 + 설명 문구 |
| 발급 이력 섹션 | 섹션 타이틀 + [전체 보기] 링크 | |
| 발급 이력 섹션 | 최근 이력 목록 | 종류 배지 / 용도 / 신청일 / 상태 뱃지 (최대 3건) |

**사용자 흐름**
- 종류 카드 탭 → `openCertModal()` 호출 → 바텀 시트 모달(M-031) 표시
- 이력 항목 탭: 진행중 → M-033 / 발급완료 → M-034 / 반려 → M-035
- [전체 보기] 탭 → 증명서 발급 이력(M-032)

**조건부 규칙**
- 발급 이력이 없을 경우: "신청한 증명서 내역이 없습니다" 안내
- 최근 이력은 최대 3건까지 표시, 이후 [전체 보기]로 유도

---

### M-031 증명서 신청

| 항목 | 내용 |
|------|------|
| **화면 목적** | 선택한 종류의 증명서를 신청한다. |
| **진입 조건** | 증명서 관리(M-030) → 종류 카드 탭 → `openCertModal()` 호출. 별도 화면 전환 없이 현재 화면 위에 바텀 시트 오버레이로 표시됨 |
| **다음 화면** | 증명서 상세 — 진행중 M-033 |

**화면 구성**

| 영역 | 컴포넌트 | 상세 |
|------|---------|------|
| 상단 핸들 바 | — | 드래그 핸들 |
| 헤더 | 종류명 소제목 + "발급 신청" 타이틀 + 닫기(×) 버튼 | |
| 입력 폼 | 증명서 명칭 | 기타증명서일 때만 표시 — **필수** / 최대 50자 / 단일 라인 |
| 입력 폼 | 용도 입력 | **필수** / 최대 100자 / 단일 라인 / placeholder "예: 은행 제출용" |
| 안내 배너 | 처리 안내 | 예상 처리 시간 + 발급 완료 시 앱 알림 안내 |
| 하단 | [발급하기] 버튼 | 전체 너비 |

**유효성 규칙**

| 오류 조건 | 처리 결과 |
|---------|---------|
| 용도 미입력 | 해당 필드 오류 표시, 신청 불가 |
| 기타증명서 명칭 미입력 | 해당 필드 오류 표시, 신청 불가 |

**조건부 규칙**
- 증명서 명칭 필드: 종류가 "기타증명서"일 때만 노출. 재직·경력증명서는 숨김
- 닫기(×) 탭 또는 배경 탭 → 모달 닫힘, M-030 유지
- 유효성 통과 후 [발급하기] 탭 → 모달 닫힘 → M-033으로 이동

---

### M-032 증명서 발급 이력

| 항목 | 내용 |
|------|------|
| **화면 목적** | 증명서 신청·발급 전체 이력을 상태별로 조회한다. |
| **진입 조건** | 증명서 관리(M-030) → [전체 보기] 링크 탭 |
| **다음 화면** | M-033 / M-034 / M-035 |

**화면 구성**

| 영역 | 컴포넌트 | 상세 |
|------|---------|------|
| 상단 내비게이션 | [← 뒤로] + "증명서 발급 이력" | 우측 버튼 없음 |
| 탭 필터 | 전체 / 진행중 / 발급완료 / 반려 | |
| 이력 목록 | 구분선형 행 (카드형 아님 · 좌우 여백 없이 전체 너비) | 종류 배지 / 용도 / 신청·발급·반려 일시 / 상태 뱃지 / 화살표 |

**조건부 규칙**
- 탭 선택 시 해당 상태 이력만 표시
- 이력이 없을 경우: 빈 상태 안내

---

### M-033 / M-034 / M-035 증명서 상세 (진행중 · 발급완료 · 반려)

> 3개 화면은 동일한 레이아웃이며, 처리 상태에 따라 표시 내용과 하단 버튼만 다르다.

**공통 화면 구성**

| 영역 | 컴포넌트 | 상세 |
|------|---------|------|
| 상단 내비게이션 | [← 뒤로] + 상태별 타이틀 | |
| 처리 상태 뱃지 | 진행중(주황) / 발급완료(초록) / 반려(빨강) | |
| 신청 정보 | 증명서 종류 / 용도 / 신청일시 / 담당부서 | |

**조건부 규칙**

| 화면 | 타이틀 | 추가 영역 | 하단 버튼 |
|------|-------|---------|---------|
| M-033 진행중 | "신청 상세" | 처리 진행 상태 타임라인 (접수→검토→발급) | 없음 (읽기 전용) |
| M-034 발급완료 | "발급 완료" | 발급일시 / 처리 담당자 / 직인 적용 여부 | [이메일로 전송] · [저장 (다운로드)] |
| M-035 반려 | "반려 상세" | 반려 사유 박스 (담당자 메시지) | [재신청] → M-030 |

> **[이메일로 전송] 탭 시**: 이메일 전송 바텀 시트 표시. 수신 이메일 직접 입력 후 전송 버튼 탭.

```
M-030 증명서 관리
  ├─ 종류 카드 탭
  │     ↓
  │   M-031 증명서 신청 (바텀 시트)

  │   M-033 진행중 상세

  └─ [전체 보기] 탭
        ↓
      M-032 증명서 발급 이력

          ↓
        M-033 / M-034 / M-035 상세
```
