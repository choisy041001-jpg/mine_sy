## 화면 상세 설계

### P-006 모바일 안내

| 항목 | 내용 |
|------|------|
| **화면 목적** | 사용자들이 모바일 앱을 설치하도록 QR·링크 안내 및 카카오 알림톡 발송을 지원한다. |
| **진입 조건** | 사이드바 하단 [모바일 안내] |
| **다음 화면** | 없음 (현재 페이지 내 처리) |

**화면 구성**

| 영역 | 컴포넌트 | 상세 |
|------|---------|------|
| 앱 설치 안내 | QR 코드 + 설치 URL | `my-insa.app/install` *(예시 URL — 실제 서비스 도메인 확정 후 교체)* |
| 앱 설치 안내 | 설치 단계 안내 | QR 스캔 → 앱 설치 → 사번 로그인 |
| 앱 설치 안내 | [QR 다운로드] 버튼 | 클릭 시 QR 코드 이미지를 **PNG 형식**으로 다운로드 |
| 앱 설치 안내 | [링크 복사] 버튼 | 클릭 시 설치 URL 클립보드 복사 |
| 사용자 설치 현황 카드 | 카드 제목 | "사용자 설치 현황" |
| 사용자 설치 현황 카드 | [설치알림이력] 버튼 | 카드 우측 상단 배치. 클릭 시 설치알림이력 팝업 오픈 |
| 사용자 설치 현황 카드 | 필터 | 전체 / 미설치만 / 설치 완료만 |
| 사용자 설치 현황 카드 | 테이블 열 구성 | 이름 / 부서 / 직급 / 연락처 / 설치 상태 / 최근 활동 / [카카오톡 발송] 버튼 |
| 사용자 설치 현황 카드 | [일괄 발송] 버튼 | 미설치 사용자 전원 대상 알림톡 일괄 발송 |
| 카카오톡 발송 팝업 | 수신자 표시 | 발송 대상 사용자 이름·연락처 표시 |
| 카카오톡 발송 팝업 | 메시지 편집 | 기본 템플릿 제공 / 최대 500자 / 다중 라인 / 카운터 표시 |
| 카카오톡 발송 팝업 | 기본 메시지 템플릿 | "안녕하세요. {회사명}에서 MY인사 사용자로 초대합니다! MY인사를 통해 어디서든 편하게 연차와 증명서 관리해보세요!" |
| 카카오톡 발송 팝업 | 미리보기 + [발송] 버튼 | |
| 설치알림이력 팝업 | 테이블 열 구성 | 발송일시 / 사용자 / 휴대폰번호 / 발송결과 (정상 / 오류) |

**사용자 흐름**
- [QR 다운로드] 클릭 → PNG 형식 QR 이미지 파일 로컬 저장
- [링크 복사] 클릭 → 설치 URL 클립보드 복사 후 토스트 "링크가 복사되었습니다." 표시
- [카카오톡 발송] 클릭 → 발송 팝업 오픈 → 메시지 확인/편집 후 [발송] 클릭 → 알림톡 발송
- [일괄 발송] 클릭 → 미설치 사용자 전원 대상 발송 팝업 오픈 → [발송] → 알림톡 일괄 발송
- [설치알림이력] 클릭 → 설치알림이력 팝업 오픈 → 발송 이력(발송일시 / 사용자 / 휴대폰번호 / 발송결과) 확인

**조건부 규칙**

**사용자 설치 현황 기준**
- 표시 대상: 그룹웨어(경리나라) 사용자 목록
- **설치 여부 판단**: 앱 로그인 기록이 한 번이라도 존재하면 **설치 완료**, 단 한 번도 없으면 **미설치**로 간주
- **최근 활동** 컬럼: 해당 사용자의 가장 최근 로그인 일시 표시. 로그인 이력이 없으면 `—` 표시

**카카오 알림톡 발송**
- 발송 방식: **카카오 알림톡** (채널 동의 불필요)
- 동일 사용자에게 재발송 허용 (횟수 제한 없음)
- 발송 성공/실패 여부는 [설치알림이력] 팝업에서 확인

**빈 상태**
- 사용자 설치 현황 테이블에 표시할 사용자가 없을 경우: **"등록된 사용자가 없습니다."** 표시
