## 화면 상세 설계

### P-004 FAQ 관리

| 항목 | 내용 |
|------|------|
| **화면 목적** | 인사 FAQ를 등록·수정·삭제 관리한다. |
| **진입 조건** | 사이드바 [FAQ 관리] |
| **다음 화면** | 없음 (현재 페이지 내 처리) |

**화면 구성**

| 영역 | 컴포넌트 | 상세 |
|------|---------|------|
| 상단 영역 | 카테고리 필터 없음 | FAQ는 단일 목록으로 관리 |
| 상단 영역 | [FAQ 등록] 버튼 | |
| FAQ 목록 테이블 | 질문 내용 / 답변 요약 / 노출 상태 / [수정] 버튼 | |
| FAQ 목록 테이블 | 노출 상태 | 활성화 ↔ 비활성화 토글 |
| FAQ 등록 팝업 | 노출 상태 / 질문 입력 / 답변 입력 | 질문: **필수** · 최대 200자 · 단일 라인 / 답변: **필수** · 최대 2,000자 · 다중 라인 · 카운터 표시 |
| FAQ 수정 팝업 | 기존 내용 불러와 편집 / [저장] · [삭제] 버튼 | 등록 팝업과 동일한 입력 제약 적용 |
| 삭제 확인 팝업 | [삭제] 클릭 시 표시 | |

**사용자 흐름**
- [+ FAQ 등록] → 등록 팝업 오픈 → 입력 후 [저장] → 목록 반영
- [수정] → 수정 팝업 오픈 → [저장] 또는 [삭제]
- [삭제] → 확인 팝업 → 확인 시 삭제 처리

**빈 상태**
- FAQ 목록에 등록된 항목이 없을 경우: **"내역이 없습니다."** 표시
