## 화면 상세 설계

### P-012 직원 프로필 관리

| 항목 | 내용 |
|------|------|
| **화면 목적** | 인사 기본 정보를 조회하고, 수정 가능한 개인 정보·인사 정보를 편집한다. |
| **진입 조건** | 사이드바 [프로필 관리] |
| **다음 화면** | 없음 (현재 페이지 내 처리) |

**화면 구성**

| 영역 | 컴포넌트 | 상세 |
|------|---------|------|
| 좌측 패널 | 프로필 사진 + 인사 기본 정보 | 사번 / 입사일 / 부서 / 직급 / 고용형태 — **읽기 전용** |
| 우측 패널 — 개인 정보 | 수정 가능 항목 | 이름(최대 30자) / 영문 이름(최대 50자, 영문·공백·하이픈만 허용) / 이메일(최대 100자, 이메일 형식 검증 — **로그인 아이디와 무관, 그룹웨어 로그인에 영향 없음**) / 연락처(최대 20자, 숫자만·하이픈 자동 삽입). **이름·연락처 등 저장 시 경리나라 사용자관리·직원관리와 자동 동기화** |
| 우측 패널 — 내 인사 정보 | 수정 가능 항목 | 자격증 / 학력 / 어학 (각 항목당 최대 100자, 다중 항목 추가 가능 — **추가 수 제한 없음**, 항목별 삭제 버튼 제공) |
| 하단 | [수정] 버튼 | 클릭 시 편집 모드 전환 |
| 편집 모드 | 수정 가능 항목 입력 필드 활성화 + 프로필 사진 변경 버튼 | [저장] · [취소] 버튼. 프로필 사진: PNG·JPG 지원 / 최대 50 MB / **88×88 px 권장** (초과·미달 이미지는 화면 맞춤 자동 조정) |

**조건부 규칙**
- 인사 기본 정보: 읽기 전용, 수정 불가
- [취소] 클릭: 변경 내용 초기화, 저장 전 원본 데이터 복원
- 프로필 사진: 편집 모드에서만 변경 가능
- **편집 중 메뉴 이탈**: 편집 모드 중 다른 사이드바 메뉴로 이동 시 별도 경고 없이 변경 내용이 저장되지 않고 초기화됨
- **경리나라 동기화**: 이름·연락처 등 개인 정보 저장 시 경리나라 사용자관리·직원관리와 즉시 동기화. 이메일은 동기화 대상이나 로그인 아이디와는 무관
