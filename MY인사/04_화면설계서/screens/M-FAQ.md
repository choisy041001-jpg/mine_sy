## 화면 상세 설계

### M-006 FAQ 목록

| 항목 | 내용 |
|------|------|
| **화면 목적** | 인사 관련 자주 묻는 질문 단일 목록을 확인한다 (카테고리 구분 없음). FAQ 등록은 관리자 PC(P-004)에서만 가능하며, 직원은 읽기 전용으로 조회만 가능하다. |
| **진입 조건** | 더보기(M-040) → [인사 FAQ] 메뉴 탭 |
| **다음 화면** | FAQ 상세 M-008 |

**화면 구성**

| 영역 | 컴포넌트 | 상세 |
|------|---------|------|
| 상단 내비게이션 | [← 뒤로] + "자주 묻는 질문" | |
| FAQ 카드 목록 | 카테고리 뱃지 | 인사FAQ (단일 분류 — 카테고리 구분 없음) |
| FAQ 카드 목록 | 질문 제목 / 작성자·날짜 | |

**사용자 흐름**
- FAQ 카드 탭 → FAQ 상세(M-008)

**조건부 규칙**
- FAQ는 인사담당자(관리자)가 PC에서 직접 등록·관리. 직원은 조회 전용
- 목록이 없을 경우: 빈 상태 안내 표시

### M-008 FAQ 상세

| 항목 | 내용 |
|------|------|
| **화면 목적** | 특정 FAQ의 질문 내용과 인사담당자 답변을 함께 확인한다. |
| **진입 조건** | FAQ 목록(M-006)에서 카드 탭 |
| **다음 화면** | 없음 |

**화면 구성**

| 영역 | 컴포넌트 | 상세 |
|------|---------|------|
| 상단 내비게이션 | [← 뒤로] + "FAQ 상세" | |
| 메타 정보 | 카테고리 뱃지 + 답변 상태 뱃지 | |
| 질문 영역 | 제목 / 내용 / 작성자·부서·작성일시 | |
| 답변 영역 | 답변완료: 답변 내용 / 답변자·부서·답변일시 | |
| 답변 영역 | 답변대기: "아직 답변이 등록되지 않았습니다" 안내 | |

**조건부 규칙**
- 답변 영역 노출 조건: 답변완료 상태일 때만 답변 내용 표시
