## 화면 상세 설계

### P-002 연차 관리

| 항목 | 내용 |
|------|------|
| **화면 목적** | 전 직원의 연차 현황을 월별로 조회하고, 결재 이력을 확인한다. 실제 승인·반려 처리는 경리나라 결재함에서 이루어진다. |
| **진입 조건** | 사이드바 [연차 관리] / 대시보드 KPI 카드 클릭 |
| **다음 화면** | 없음 (현재 페이지 내 처리) |

**화면 구성**

| 영역 | 컴포넌트 | 상세 |
|------|---------|------|
| 상단 툴바 | 연도 선택 | **기본값: 오늘 날짜 기준 현재 연도(yyyy).** 드롭다운 변경 시 해당 연도의 연차 현황으로 즉시 필터링. **과거·미래 연도 제한 없이** 자유롭게 연도를 선택하여 연차 현황 조회 및 관리 가능 |
| 상단 툴바 | 부서 선택 필터 | 드롭다운 / 전체 + 사업장 내 부서 목록 |
| 상단 툴바 | 직원명 검색 | 최대 50자 / 1자 이상 입력 시 실시간 필터링 (300 ms 디바운스) / × 버튼으로 초기화 |
| 상단 툴바 | [일반근로자 불러오기] 버튼 | 경리나라 직원관리에 등록된 일반근로자 중 **조회 시점 재직 상태인 직원**만 불러옴. 일괄 등록 팝업 오픈. **현재 선택된 연도에만 저장되며, 다른 연도에는 영향을 주지 않음** |
| 상단 툴바 | [엑셀 다운로드] 버튼 | 현재 선택 연도 기준 화면에 표시된 목록 전체 다운로드. 포함 항목: 직원명 / 부서 / 총 연차 / 1월~12월 월별 사용 현황 / 잔여 연차 |
| 안내 문구 | 테이블 상단 | "총연차를 클릭하시면 연차를 등록할 수 있습니다" |
| 연차 현황 테이블 | 열 구성 | 직원명 / 부서 / 총 연차 / 사용 / 1월~12월 / 잔여 |
| 연차 현황 테이블 | 직원명 | 클릭 가능 → 우측 상세 패널 열림 |
| 연차 현황 테이블 | 총 연차 셀 | 더블클릭 시 직접 편집 가능 (**소수점 입력 허용**, 엔터/포커스 이탈 시 저장) |
| 연차 현황 테이블 | 월별 셀 | 마우스 오버 시 해당 월 사용 이력 툴팁 표시 |
| 우측 상세 패널 | 직원 기본 정보 + 연차 통계 | 직원명 클릭 시 슬라이드 오픈 |
| 우측 상세 패널 | 결재 이력 목록 | **현재 선택된 연도의 이력만 표시**. 연도 전환 시 해당 연도의 결재 이력으로 갱신 |

**사용자 흐름**
- 직원명 클릭 → 우측 상세 패널 오픈
- [일반근로자 불러오기] 클릭 → 경리나라 직원관리 일반근로자 중 조회 시점 재직 상태인 직원 목록 조회 → 직원 일괄 등록 팝업 오픈

**조건부 규칙**
- 결재 이력 없는 직원 패널: "이력 없음" 안내 표시
- 직원 불러오기 모달 — 미등록 직원이 없을 경우: **"등록할 직원이 없습니다."** 표시
- **테이블 로딩 방식**: 직원 수가 많을 경우 무한 스크롤 방식으로 추가 로드. 페이지네이션 없음.
- **연도 기본값**: 화면 최초 진입 시 오늘 날짜 기준 현재 연도(yyyy)로 자동 설정
- **일반근로자 불러오기 대상**: 경리나라 직원관리 등록 일반근로자 중 **조회 시점 재직 상태**인 직원만 목록에 표시됨
- **일반근로자 불러오기 연도 범위**: 등록된 직원 데이터는 현재 선택된 연도에만 귀속됨. 연도를 전환해도 해당 연도 데이터에 영향 없음
- **엑셀 다운로드 범위**: 현재 선택 연도·적용 중인 부서·직원명 필터 기준으로 화면에 표시된 목록과 동일한 데이터(직원명 / 부서 / 총 연차 / 1월~12월 월별 사용 현황 / 잔여 연차) 다운로드

**데이터 동기화 규칙**
- 경리나라 **인사급여** 이용 고객사의 경우, 직원별 연차 데이터(총 연차·사용·잔여)는 인사급여의 연차와 자동 동기화된다.
- 인사급여 연동 고객사는 총 연차 셀 더블클릭 수동 편집이 비활성화된다 (인사급여 데이터가 원천).
- 인사급여 미사용 고객사는 총 연차 셀 더블클릭 수동 편집 방식을 그대로 유지한다.

**연차 차감 시점**
- 실제 승인·반려 처리는 경리나라 결재함 앱에서 이루어진다.
- MY인사는 경리나라 결재함에서 **결재가 최종 완료된 시점**에 해당 직원의 연차를 차감한다.
- 결재 대기 상태(진행중)인 신청 건은 연차가 차감되지 않으며, 잔여 연차에서 제외되지 않는다.
