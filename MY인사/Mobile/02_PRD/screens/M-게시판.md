## 데이터 표시 기준

> M-006/M-008(인사 FAQ)의 데이터 표시 기준은 **더보기_알림** 문서에서 정의한다. 본 섹션은 게시판(M-002~M-005)을 다룬다.

**① 데이터 출처**
- 게시판·게시글·댓글은 **그룹웨어 게시판**에서 관리되며, 그룹웨어 앱은 API로 연동해 **현재 로그인한 사업장(그룹웨어)에서 본인이 속한(접근 권한이 있는) 게시판의 게시글만** 표시한다(타 사업장·미소속 게시판 미노출). 게시판 생성·삭제 등 운영 관리는 그룹웨어 게시판에서 수행한다. 게시판 목록은 API 동적 로드한다.

**② M-002 게시판 홈 (`board`)**
- 타이틀 `게시판` + 부제 `우리 회사 소식과 자유로운 대화`(필터 적용 시 게시판명으로 대체).
- 필터(≡): 게시판 선택 바텀 시트(`전체`+운영 게시판), 적용 중이면 아이콘 강조. 알림(🔔): 미읽음 1건 이상 시 빨간 점 뱃지, 전체 읽음 시 소멸.
- 피드 행: `제목`(**신규 + 미확인 글이면** 제목 옆 `N` 뱃지, **상세 열람 시 소멸**) / `{게시판명} · {작성자}` / `{M월 D일}`. 빈 상태 `등록된 게시글이 없습니다.`. FAB(+) → `compose`. *(댓글에는 N 뱃지 없음)*

**③ M-003 게시글 목록 (`post-list`)**
- 상단 게시판명. 행: `제목`(공지 글 `공지` 핀·상단 고정) + `{작성자} · {M월 D일} · 댓글 {N}`. 인사 FAQ 채널은 글쓰기 FAB 숨김.

**④ M-004 게시글 상세 (`post-detail`)**
- 본문: `제목`(공지 핀) / `{작성자} · {등록일시 YYYY.MM.DD HH:MM}` / 본문 / **첨부파일 목록**(본문 최하단, `파일명.확장자` + 파일/이미지 아이콘, 탭 시 다운로드, 이미지 포함). **조회수 미표시.**
- 댓글: 헤더 `댓글 {N}`, 각 댓글 = **이름(이름만) + 내용 + 작성시간**(부서·아바타 미표시 — M-000 §5.2). 내 댓글은 더보기(···) → 수정/삭제. 입력창 최대 300자. **등록/수정 시 내용 필수** — 비어 있으면 토스트 `내용을 입력해주세요.` 후 중단, 입력값 있으면 등록/수정 처리(등록 시 작성자=현재 사용자, 입력창 초기화). **태블릿 분할 중 글 수정 저장·작성 취소 후 상세 복귀 시 분할 유지.**
- 본인 글 더보기 → 수정(`post-edit` pre-fill)/삭제 확인 팝업.

**⑤ M-005 글 작성 (`compose`)**
- 게시판 선택(필수, 인사 FAQ 제외) / 제목(필수·100자) / 본문(필수·3,000자·카운터) / 파일 첨부(탭 시 **첨부 방식 선택 바텀시트**: 카메라 촬영하기=디바이스 카메라 / 앨범에서 선택하기=디바이스 갤러리 / 파일 선택하기=디바이스 파일(폴더). 이미지 JPG·PNG + 문서 PDF·DOC·DOCX·XLS·XLSX·PPT·PPTX·다건 첨부·최대 5개·파일당 10 MB·**총 합계 50 MB 이내**·**첨부 시 [첨부] 버튼 하단 첨부파일 목록 표시(`파일명.확장자` + 파일/이미지 아이콘, 우측 × 개별 제거)**).
- [등록] → 검증 후 `✅ 게시글이 등록되었습니다` + `board` 복귀. **등록 글·신규 댓글의 작성 시각은 등록 시점을 `YYYY.MM.DD HH:MM` 형식으로 표시**(작성자=현재 사용자). [취소] → 입력 있으면 `작업을 취소하시겠습니까?` 확인.
