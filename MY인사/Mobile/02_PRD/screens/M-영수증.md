## 데이터 표시 기준

> 화면별 표시 데이터의 출처·형식·연동 결과를 필드 단위로 정의한다. 영수증·결의서 데이터는 경리나라와 실시간 연동된다.

### 공통 — 데이터 출처
- **영수증**: 등록 즉시 경리나라 **매입(간이영수증)**에 자동 저장. 지급수단에 따라 추가 장부(현금출납장·전도금원장) 연동.
- **거래처**: 경리나라 거래처 마스터(명칭·사업자번호). **직원**: 경리나라 직원관리. 사용처·지급처 검색 레이어 데이터 출처.
- **금액**: 원 단위, 3자리 콤마(예: `25,000`), 0원 등록 불가.
- **전자결재 동작**: 경리나라 지출결의서 전자결재 설정(ON/OFF)에 따라 버튼명·결재선·전송 경로 결정.

### 지급수단별 경리나라 연동 매핑
| 지급수단 | 설명 | 경리나라 자동 연동 |
|---------|------|------------------|
| **개인 경비** (기본값) | 개인카드·현금 등 본인 부담 지출 | 매입(간이영수증) 등록 |
| **회사 현금으로 지급** | 회사 현금을 수령해 지출 | 매입(간이영수증) + 현금출납장 자동 등록 |
| **전도금으로 지급** | 미리 지급받은 전도금으로 지출 | 매입(간이영수증) + 전도금원장 자동 등록 |

### M-020 영수증 관리
- 날짜 헤더: 사용일시 날짜(YYYY.MM.DD) 그룹핑. 카드: 사용처·용도·지급처·금액.
- '결의서 있음' 뱃지: 결재요청(진행중)/결재완료(승인) 결의서 포함 시 표시.
- 하단 버튼명: 항상 `[결의서 작성 (N건)]`(전자결재 ON/OFF 무관, N=선택 건수). 전자결재 분기는 M-023에서만 처리.

### M-021 영수증 등록
- **사용일시(필수)**: `YYYY.MM.DD HH:mm` 표시. 바텀시트 = 상단 달력(월 이동·날짜 그리드, 과거·미래 모두 선택 가능) + 하단 아날로그 시계(24시간제: 시 모드 바깥 1~12시/안쪽 13~24시[24=00 표기], 분 모드 5분 단위). 시 선택 시 분 모드 자동 전환.
- **금액(필수)**: 원·콤마·최대 12자리. **사용처(필수)**: 직접입력(명칭+사업자번호 선택)/거래처 검색.
- **용도(선택)** 드롭다운, **내용(선택)**, **지급처(선택)**: 직원/거래처 검색·기타 직접입력.
- **지급수단(필수)**: 위 매핑표 참조 — 선택값에 따라 연동 장부 결정.

### M-022 영수증 상세
- 상세 정보: 사용일시(`YYYY.MM.DD HH:mm`)·사용처·용도·지급처·지급수단·사용금액(등록값, 경리나라 매입 기준). 처리 이력 타임라인(등록→결의서 포함→승인).
- 하단 마지막 버튼: 항상 `[결의서 작성]`(전자결재 ON/OFF 무관). 제출 버튼명 분기는 M-023에서 처리.

### M-023 결의서 작성 / 결재요청
- 타이틀/버튼: 전자결재 ON → "결재요청"·`[상신하기]`(경리나라 결재함 상신, 결과 푸시 M-050) / OFF → "결의서 작성"·`[저장하기]`(지출결의서관리 전송). *(프로토타입은 우측 상단 미리보기 토글로 ON/OFF 두 상태 확인 가능 — 실서비스는 경리나라 설정값 고정.)*
- 결재선: **전자결재 ON일 때만 표시**. OFF에서는 결재선 섹션 미노출.
- 영수증 내역(사용일시·사용처·용도·금액 읽기 전용) + 합계 자동 합산.
- 결재선: 전자결재 ON 시만 표시, 경리나라 자동 지정. 2개 이상 시 `[결재선 변경]` 라디오 팝업.
