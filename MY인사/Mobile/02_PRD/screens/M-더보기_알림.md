## 앱 푸시 알림 명세

관리자가 PC에서 연차 결재 또는 증명서 발급을 처리하는 즉시, 해당 직원의 모바일 앱으로 푸시 알림이 발송됩니다.

### 공통 동작 규칙

| 항목 | 규칙 |
|------|------|
| **발송 시점** | 관리자 처리 완료(승인·발급·반려) 또는 시스템 이벤트 발생 즉시 서버에서 발송 |
| **수신 대상** | 신청한 직원 1인 (공지사항 알림은 해당 사업장 전체 직원) |
| **앱 포그라운드** | 인앱 알림 배너 (화면 상단 슬라이드다운, 자동 소멸) |
| **앱 백그라운드·종료** | OS 푸시 알림 (기기 알림 센터에 적재) |
| **탭 동작(인앱 목록)** | M-050 알림 행 탭 시 해당 기능 메인 화면(연차 관리·증명서 관리·영수증 관리·게시판 홈)으로 이동 |
| **탭 동작(기기 알림 딥링크)** | 기기 알림 탭 시 자동로그인 설정·알림의 사용자/사업장 일치 여부에 따라 앱 진입을 분기(아래 "기기 알림 탭 → 앱 진입 분기" 표). 일치 시 알림 메뉴(M-050) 이동, 불일치 시 게시판 홈(M-002) 유지. 이미 로그인된 상태에서 다른 계정/사업장 알림 탭 시 강제 로그아웃 → 로그인 화면(M-001) |
| **미수신 처리** | 앱 알림 권한이 꺼진 경우 발송 시도 후 무시 (재발송 없음) |
| **방해금지 모드** | M-051에서 설정한 시간대에는 알림 수신 차단. 해당 시간 종료 후 미수신 알림을 일괄 표시 |
| **알림 설정 OFF** | M-051에서 푸시 알림 받기를 OFF한 경우 **디바이스(OS) 푸시 발송만 차단**(앱 내 알림 목록 M-050에는 수신 알림이 계속 적재됨) |
| **항목별 설정 OFF** | M-051의 항목별 토글이 OFF인 유형은 해당 유형 알림만 차단 |

### 기기 알림 탭 → 앱 진입 분기

> 기기 알림에는 **수신 대상(사용자 + 사업장)** 과 **딥링크 목적지(알림 메뉴 M-050)** 가 포함된다. 알림 탭 시 앱 진입·로그인 처리는 **자동로그인 설정 여부**와 **'알림의 사용자/사업장' == '앱에 로그인되는 사용자/사업장' 일치 여부**에 따라 분기한다. **알림 메뉴(M-050) 자동 이동은 두 정보가 일치할 때만** 수행하고, 불일치 시 게시판 홈(M-002, 로그인 첫 화면)을 유지한다. (로그인 흐름 → M-001)

| # | 앱 상태 / 조건 | 앱 진입 처리 | 이동 화면 |
|---|------|------|------|
| 1 | 자동로그인 설정됨 + 알림 사업장 = 자동로그인 사업장 | 해당 계정/사업장 **자동 로그인** | **M-050 알림 메뉴** |
| 2 | 자동로그인 미설정 → 알림과 **동일한** 사용자/사업장으로 로그인 | 앱 실행 → 로그인(M-001) | **M-050 알림 메뉴** |
| 3 | 자동로그인 미설정 → 알림과 **다른** 계정/사업장으로 로그인 | 앱 실행 → 로그인(M-001) | 게시판 홈(M-002) 유지(이동 안 함) |
| 4 | **이미 로그인된 상태**에서 **다른 계정/사업장** 알림 탭 | **강제 로그아웃 → 로그인 화면(M-001)** | (재로그인 결과에 따름 ↓) |
| 4-1 | └ 알림과 **동일한** 사용자/사업장으로 재로그인 | 로그인(M-001) | **M-050 알림 메뉴** |
| 4-2 | └ 알림과 **다른** 사용자/사업장으로 재로그인 | 로그인(M-001) | 게시판 홈(M-002) 유지 |

- 이미 로그인된 상태에서 **동일 계정/사업장** 알림 탭 시: 로그아웃 없이 곧바로 **M-050 알림 메뉴**로 이동(4번은 계정/사업장 불일치일 때만 강제 로그아웃).
- 딥링크 목적지는 개별 상세가 아니라 **알림 메뉴(M-050)** 이며, 이후 알림 행 탭 시 이동은 "알림 문구 명세"/M-050 정의를 따른다.

### 알림 문구 명세

기기 알림 센터·잠금화면에 표시되는 실제 알림 문구입니다. OS 알림 구조상 앱 이름("그룹웨어")은 시스템이 자동으로 표시하며, 아래 제목·본문만 개발에서 제어합니다.

| 이벤트 | 제목 (title) | 본문 (body) | 표시 예시 |
|--------|------------|-----------|---------|
| 연차 신청 **승인** | 휴가신청서 결재가 승인되었습니다 | `{신청일자} {휴가종류}` 휴가 신청을 `{결재자}`님이 승인하였습니다. | 5/26~5/27 연차 휴가 신청을 홍길동님이 승인하였습니다. |
| 연차 신청 **반려** | 휴가신청서 결재가 반려되었습니다 | `{신청일자} {휴가종류}` 휴가 신청이 반려되었습니다. 사유를 확인해 주세요. | 5/26 오전 반차 휴가 신청이 반려되었습니다. 사유를 확인해 주세요. |
| 증명서 발급 **완료** | 증명서 발급이 완료되었습니다 | 신청하신 `{증명서종류}`가 발급되었습니다. 확인 후 저장하세요. | 신청하신 재직증명서가 발급되었습니다. 확인 후 저장하세요. |
| 증명서 발급 **반려** | 증명서 발급이 반려되었습니다 | `{증명서종류}` 발급 신청이 반려되었습니다. 사유를 확인해 주세요. | 재직증명서 발급 신청이 반려되었습니다. 사유를 확인해 주세요. |
| 새 **공지사항** 게시 | 새 공지사항이 등록되었습니다 | `{게시판명}`에 새 게시글이 등록되었습니다. | 경영지원부서 공지에 새 게시글이 등록되었습니다. |
| 결의서 결재 **완료** | 결의서 결재가 승인되었습니다 | 제출하신 `{결의서제목}` 결의서가 최종 승인되었습니다. | 경비 결의서가 최종 승인되었습니다. |
| 결의서 결재 **반려** | 결의서 결재가 반려되었습니다 | 제출하신 `{결의서제목}` 결의서가 반려되었습니다. 사유를 확인해 주세요. | 경비 결의서가 반려되었습니다. 사유를 확인해 주세요. |

> **치환자 규칙**
> - `{신청일자}`: 단일 날짜 `5/26`, 범위 `5/26~5/27`
> - `{휴가종류}`: `연차` / `오전 반차` / `오후 반차`
> - `{결재자}`: 결재자 이름 (예: `홍길동`)
> - `{증명서종류}`: `재직증명서` / `경력증명서` / `기타증명서`
> - `{게시판명}`: 게시글이 등록된 게시판 이름
> - `{결의서제목}`: 결의서 작성 시 입력한 제목
