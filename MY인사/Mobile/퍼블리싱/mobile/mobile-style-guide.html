탭바

화면 하단 고정 내비게이션. .app 컨테이너 안에서 position:absolute;bottom:0으로 배치된다. 탭바 높이(64px)만큼 콘텐츠 하단에 .pb-tabbar(padding-bottom:80px)를 추가해야 한다.

주의: .tabbar는 .app(position:relative) 기준으로 position:absolute;bottom:0이다. 탭바 위 콘텐츠 영역에 반드시 .pb-tabbar를 추가한다.

화면 구조

앱 전체 레이아웃 구조. .phone > .app > .screen.active 계층 구조.

인라인 style="display:none" — 의도적으로 남겨둔 것
화면 전환·오버레이 토글은 JS가 element.style.display = 'flex' / '' 방식으로 직접 제어한다. 인라인 스타일이 CSS 클래스보다 우선순위가 높기 때문에, 이 패턴에서 display:none은 반드시 인라인에 있어야 한다. CSS 파일로 빼거나 클래스로 대체하면 JS 토글이 동작하지 않는다.
앱 레이아웃 구조 .phone .app .screen.active
HTML 구조
<div class="phone">           <!-- 앱 루트 컨테이너 (height:100%) -->
  <div class="app">            <!-- flex col, position:relative -->

    <div class="screen active" data-screen="home">
      <!-- 콘텐츠 + .pb-tabbar 로 탭바 높이 확보 -->
    </div>

    <div class="screen" data-screen="approval">…</div>

    <!-- 탭바: position:absolute;bottom:0 -->
    <div class="tabbar">…</div>

    <!-- 바텀시트: position:absolute;inset:0 -->
    <div class="sheet-backdrop" id="sheet-leave">
      <div class="sheet">…</div>
    </div>

  </div>
</div>
클래스설명
.phone앱 루트 컨테이너. height:100%로 전체화면 채움
.app앱 콘텐츠 컨테이너. flex col, position:relative
