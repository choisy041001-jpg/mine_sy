## 1. 문서 개요

### 1.1 목적

본 문서는 중소기업용 통합 HR 솔루션 「MY인사」의 소프트웨어 요구 사항을 정의한다. 개발팀, 디자인팀, QA팀이 공통 기준으로 사용하며, 설계·구현·테스트의 근거로 활용된다.

### 1.2 범위

MY인사는 두 가지 클라이언트로 구성된다.

- **모바일 웹앱 (375 px 기준) — 직원용**: 게시판, 연차 신청/결재, 영수증 등록 및 결의서 작성, 증명서 신청, FAQ, 더보기(내 정보)
- **PC 웹 SPA — 관리자·직원 통합**: 관리자 대시보드, 연차 관리, 업무신청관리(증명서), FAQ 관리, 직원용 대시보드, 업무신청, 프로필 관리, MY인사 소개, 모바일 설치 안내

### 1.3 용어 정의

| 용어 | 정의 |
|------|------|
| 직원 (Employee) | 모바일 앱을 주로 사용하는 일반 임직원. 사번 또는 이메일로 로그인. |
| 관리자 (Admin) | 인사담당자·경리담당자. PC에서 연차·증명서·FAQ를 관리. |
| 결재선 | 경리나라에서 자동 지정되는 결재 경로. 사용자가 직접 수정 불가. |
| 경리나라 | WEBCASH의 회계/경비 관리 SaaS. 영수증 매입 및 결재선 정보 제공. |
| 결의서 | 영수증 1건 이상을 묶어 경비 지출을 승인받는 전자 문서. |
| FAB | Floating Action Button. 화면 우하단 고정형 주요 액션 버튼. |
| SPA | Single Page Application. 페이지 전환 없이 JS 라우팅으로 화면을 교체. |
| 연차사용계획서 | 분기·월 단위로 예정 연차를 미리 신청하는 문서. |

### 1.4 참고 자료

- `PRD_클로드.md` — MY인사 제품 요구 사항 문서 (원문)

## 2. 제품 개요

### 2.1 제품 한 줄 정의

MY인사는 연차 신청·결재, 증명서 발급, 영수증 기반 경비 결의서 작성 등 인사·경비 업무를 직원(모바일)과 인사담당자(PC)에게 제공하는 중소기업용 HR 솔루션이다.

### 2.2 사용 시나리오

1. **직원 시나리오**: 직원이 외부에서 경비 영수증을 촬영·등록하고 결의서를 상신하거나, 연차를 신청하거나, 증명서 발급을 요청할 때 모바일 MY인사를 사용한다.
2. **관리자 시나리오**: 인사담당자가 직원의 연차 현황 확인·결재 처리, 증명서 발급, FAQ 관리 등 인사업무를 처리할 때 PC MY인사를 사용한다.

### 2.3 타겟 사용자 및 페인 포인트

| 페르소나 | 대상 | 핵심 페인 포인트 |
|---------|------|----------------|
| 페르소나 1 | 10인 미만 회사 직원 | ① 경비 영수증을 카톡·실물로 경리에 제출해야 함<br>② 연차 현황 확인을 위해 인사담당자에게 별도 문의<br>③ 증명서 발급 요청·진행 현황 파악 불편 |
| 페르소나 2 | 인사담당자·경리담당자 | ① 카톡·전화로 연차 문의 받으면 엑셀로 수기 관리<br>② 영수증을 카톡·실물로 받아 풀로 붙여 관리<br>③ 증명서 카톡 발급으로 미처리 업무 파악 어려움 |

### 2.4 성공 기준

- 직원이 외부에서도 쉽고 편리하게 인사업무 신청 및 영수증 관리를 수행할 수 있다.
- 인사담당자·경리담당자가 정확하고 편리하게 업무를 처리 및 관리할 수 있다.

## 3. 시스템 아키텍처 및 기술 스택

### 3.1 기술 스택

| 구분 | 기술 | 비고 |
|------|------|------|
| Frontend (모바일) | 모바일 웹 SPA | 375 px 기준, 반응형 |
| Frontend (PC) | 웹 기반 SPA | 사이드바(240 px) + 메인 레이아웃 |
| 디자인 시스템 | Pretendard / Malgun Gothic | Accent `#2363A8`, 라이트 테마 |
| 외부 연동 — 경리나라 | REST API (WEBCASH 내부) | 결재선 자동 지정, 영수증 매입 자동 저장 |
| 외부 연동 — 카카오톡 | 카카오 알림톡 API | 앱 설치 안내 메시지 발송 |
| 모바일 설치 URL | `my-insa.app/install` | QR 코드 및 직링크 제공 |

### 3.2 디자인 토큰

| 토큰 | 값 | 용도 |
|------|----|------|
| `--accent` | `#2363a8` | 주요 강조색 (버튼, 링크, 활성 상태) |
| `--accent-light` | `#eff5fa` | 강조색 배경 (선택된 메뉴, 뱃지 배경) |
| `--success` | `#16a34a` | 승인 · 발급완료 상태 |
| `--warning` | `#d97706` | 진행중 · 대기 상태 |
| `--danger` | `#dc2626` | 반려 · 오류 상태 |
| `--text` | `#111118` | 본문 텍스트 |
| `--text-secondary` | `#4a4a5a` | 보조 텍스트 |

### 3.3 외부 연동 개요

- **경리나라 연동**: 영수증 등록 시 경리나라 매입(간이영수증)에 자동 저장. 연차·경비 결의서 결재선은 경리나라 기준으로 자동 지정 (사용자 수정 불가). 경리나라 **인사급여** 이용 고객사의 경우, PC 연차 관리의 직원별 연차 데이터는 인사급여의 연차와 자동 동기화된다.
- **카카오톡 연동**: PC 모바일 안내 화면에서 관리자가 미설치 직원에게 앱 설치 링크를 개별 또는 일괄 발송.

| **사용자 액션** | 수정 버튼 → 편집 모드 / 프로필 사진 카메라 아이콘 → 파일 선택 후 사진 변경 / 저장 버튼 → 저장 후 조회 모드 복귀 / 취소 버튼 → 변경 취소 후 조회 모드 복귀 / 인사 정보 항목 수정 아이콘 → 수정 모달(항목 편집 또는 삭제) |
| **다음 화면** | 없음 (현재 페이지 내 처리) |

## 8. 범위 외 (Out of Scope)

아래 기능은 이번 버전에 포함하지 않는다. 기획·개발·QA 단계에서 해당 항목은 구현 대상에서 제외한다.

| 제외 항목 | 사유 |
|---------|------|
| 급여 명세서 조회/발급 | 경리나라 연동으로 처리 — MY인사 자체 급여 기능 불포함 |
| 근태 관리 (출퇴근 기록) | 출퇴근 체크인/아웃, GPS 연동 등 근태 기능 전체 제외 |
| 조직도 / 인사발령 관리 | 직원 이동·승진·발령 등 인사 마스터 관리 제외 |
| 관리자 PC 게시판 관리 | 게시글 작성·삭제·관리는 모바일(직원)만 구현 |
| 경리나라 외 회계 시스템 연동 | 경리나라 이외 ERP/회계 솔루션 연동 불포함 |
| 결재선 수동 편집 | 결재선은 경리나라 기준 자동 지정 — 사용자 직접 변경 불가 |

## 9. 완성 기준 (Acceptance Criteria)

### 9.1 모바일 — 직원용

- [ ] 사번/이메일로 로그인 후 사업장 선택 팝업에서 사업장 선택 → 게시판 화면 진입
- [ ] 아이디 찾기 / 비밀번호 변경 (휴대폰·이메일 인증)
- [ ] 사업장 전환 팝업에서 사업장 선택 후 전환
- [ ] 게시판 피드 확인, 게시글 목록·상세 조회, 댓글 작성, 글 작성
- [ ] 연차 관리 화면에서 잔여 연차 및 결재 현황(진행중/승인/반려) 카드 확인
- [ ] 캘린더 날짜 드래그로 연차 신청 및 결재 상신
- [ ] 연차 취소 신청 (날짜·휴가 종류·취소 사유 선택) 및 연차사용계획서 제출
- [ ] 연차 사용내역 전체/종일/반차 필터 조회
- [ ] 영수증 이미지 첨부 및 정보 입력 후 등록 (경리나라 자동 저장)
- [ ] 영수증 선택 후 결의서 작성 및 상신
- [ ] 재직/경력/기타 증명서 신청 (바텀 시트 모달) 및 처리 상태(진행중/발급완료/반려) 추적
- [ ] 발급완료 증명서 PDF 저장 또는 이메일 전송
- [ ] 인사 FAQ 열람 (카테고리별 아코디언, 상세 조회)
- [ ] 더보기에서 내 정보(인사 마스터) 조회

### 9.2 PC — 관리자용

- [ ] 대시보드 KPI 카드 확인 (대기 연차, 미답변 FAQ, 미처리 증명서, 총 직원수)

- [ ] 연차 관리: 직원별 연차 현황 및 월별 사용 이력 조회, 총연차 더블클릭 수정, 결재 처리
- [ ] 업무신청관리: PDF 첨부 후 발급 완료 처리 및 반려 처리
- [ ] FAQ 관리: FAQ 등록·수정·삭제 및 노출 상태 관리
