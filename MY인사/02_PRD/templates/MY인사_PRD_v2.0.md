## 1. What — 한 줄 정의

MY인사는 연차 신청·결재, 증명서 발급, 영수증 기반 경비 결의서 작성 등 인사·경비 업무를 직원(모바일)과 인사담당자(PC)에게 제공하는 **중소기업용 HR 솔루션**입니다.

---

## 2. 메인 사용 시나리오

1. **직원 시나리오**: 직원이 외부에서 경비 영수증을 촬영·등록하고 결의서를 상신하거나, 연차를 신청하거나, 증명서 발급을 요청할 때 **모바일 MY인사**를 사용한다.
2. **관리자 시나리오**: 인사담당자가 직원의 연차 현황 확인·결재 처리, 증명서 발급, FAQ 관리 등 인사업무를 처리할 때 **PC MY인사**를 사용한다.

---

## 3. Who — 타겟 사용자

### 페르소나 1: 직원 (10인 미만 회사)

- **핵심 페인 포인트 1**: 외부에서 사용한 경비 영수증을 카톡이나 실물로 경리 직원에게 제출해야 하는 불편
- **핵심 페인 포인트 2**: 본인의 연차 현황을 파악하기 위해서는 인사담당자에게 별도 문의를 해야 하는 불편
- **핵심 페인 포인트 3**: 기관에 제출해야 하는 증명서 발급 요청을 카톡으로 하고, 진행현황을 확인하기 어려운 불편

### 페르소나 2: 인사담당자·경리담당자 (10인 미만 회사)

- **핵심 페인 포인트 1**: 직원이 카톡이나 전화로 연락하면 엑셀을 통해 수기로 연차 관리 및 확인해줘야 하는 불편
- **핵심 페인 포인트 2**: 직원이 외부에서 사용한 영수증을 카톡이나 실물로 전달받아 풀로 붙이고 관리해야 하는 불편
- **핵심 페인 포인트 3**: 증명서 발급 시 카톡으로 보내주다 보니 미처리된 업무 파악에 어려움

### 성공 정의

- 직원이 외부에서도 쉽고 편리하게 인사업무 신청 및 영수증 관리를 수행할 수 있다.
- 인사담당자·경리담당자가 정확하고 편리하게 업무를 처리 및 관리할 수 있다.

## 4. 시스템 아키텍처 및 기술 스택

### 4.1 기술 스택

| 구분 | 기술 | 비고 |
|------|------|------|
| Frontend (모바일) | 모바일 웹 SPA | 375 px 기준, 반응형 |
| Frontend (PC) | 웹 기반 SPA | 사이드바(240 px) + 메인 레이아웃 |
| 디자인 시스템 | Pretendard / Malgun Gothic | Accent `#2363A8`, 라이트 테마 |
| 외부 연동 — 경리나라 | REST API (WEBCASH 내부) | 결재선 자동 지정, 영수증 매입 자동 저장 |
| 외부 연동 — 카카오톡 | 카카오 알림톡 API | 앱 설치 안내 메시지 발송 |

### 4.3 외부 연동 개요

- **경리나라 연동**: 영수증 등록 시 경리나라 매입(간이영수증)에 자동 저장. 연차·경비 결의서 결재선은 경리나라 기준으로 자동 지정 (사용자 수정 불가). 경리나라 **인사급여** 이용 고객사의 경우, PC 연차 관리의 직원별 연차 데이터는 인사급여의 연차와 자동 동기화된다. PC 연차 관리의 **일반근로자 불러오기** 기능은 경리나라 **직원관리**의 일반근로자 목록을 불러온다.
- **카카오톡 연동**: PC 모바일 안내 화면에서 관리자가 미설치 직원에게 앱 설치 링크를 개별 또는 일괄 발송.

### 4.4 앱 푸시 알림

관리자가 PC에서 연차 결재 또는 증명서 발급을 처리하는 즉시, 해당 직원의 모바일 앱으로 푸시 알림이 발송된다. 직원은 모바일 앱 내 알림 목록(M-050)에서 수신 이력을 확인하고, 알림 설정(M-051)에서 수신 여부 및 방해금지 모드를 제어할 수 있다.

#### 공통 동작 규칙

| 항목 | 규칙 |
|------|------|
| **발송 시점** | 관리자 처리 완료(승인·발급·반려) 또는 시스템 이벤트 발생 즉시 서버에서 발송 |
| **수신 대상** | 신청한 직원 1인 (공지사항 알림은 해당 사업장 전체 직원) |
| **앱 포그라운드** | 인앱 알림 배너 (화면 상단 슬라이드다운, 자동 소멸) |
| **앱 백그라운드·종료** | OS 푸시 알림 (기기 알림 센터에 적재) |
| **탭 동작** | 푸시 탭 시 해당 결과 상세 화면으로 딥링크 이동. 미로그인 상태이면 로그인 후 이동 |
| **미수신 처리** | 앱 알림 권한이 꺼진 경우 발송 시도 후 무시 (재발송 없음) |
| **방해금지 모드** | M-051에서 설정한 시간대에는 알림 수신 차단. 해당 시간 종료 후 미수신 알림을 일괄 표시 |
| **알림 설정 OFF** | M-051에서 푸시 알림 받기를 OFF한 경우 모든 유형 발송 차단 |
| **항목별 설정 OFF** | M-051의 항목별 토글이 OFF인 유형은 해당 유형 알림만 차단 |

#### 푸시 알림 문구 및 명세

기기 알림 센터·잠금화면에 표시되는 실제 알림 문구는 다음과 같다. OS 알림 구조상 앱 이름("MY인사")은 시스템이 자동으로 표시하며, 아래 제목·본문만 개발에서 제어한다.

| 이벤트 | 제목 (title) | 본문 (body) | 표시 예시 |
|--------|------------|-----------|---------|
| 연차 신청 **승인** | 휴가신청서 결재가 승인되었습니다 | `{신청일자} {휴가종류}` 휴가 신청을 `{결재자}`님이 승인하였습니다. | 5/26~5/27 연차 휴가 신청을 홍길동님이 승인하였습니다. |
| 연차 신청 **반려** | 휴가신청서 결재가 반려되었습니다 | `{신청일자} {휴가종류}` 휴가 신청이 반려되었습니다. 사유를 확인해 주세요. | 5/26 오전 반차 휴가 신청이 반려되었습니다. 사유를 확인해 주세요. |
| 증명서 발급 **완료** | 증명서 발급이 완료되었습니다 | 신청하신 `{증명서종류}`가 발급되었습니다. 확인 후 저장하세요. | 신청하신 재직증명서가 발급되었습니다. 확인 후 저장하세요. |
| 증명서 발급 **반려** | 증명서 발급이 반려되었습니다 | `{증명서종류}` 발급 신청이 반려되었습니다. 사유를 확인해 주세요. | 재직증명서 발급 신청이 반려되었습니다. 사유를 확인해 주세요. |
| 새 **공지사항** 게시 | 새 공지사항이 등록되었습니다 | `{게시판명}`에 새 게시글이 등록되었습니다. | 경영지원부서 공지에 새 게시글이 등록되었습니다. |

> **치환자 규칙**
> - `{신청일자}`: 연차 신청 기간. 단일 날짜면 `5/26`, 범위면 `5/26~5/27`
> - `{휴가종류}`: `연차` / `오전 반차` / `오후 반차`
> - `{결재자}`: 결재자 이름 (예: `홍길동`)
> - `{증명서종류}`: `재직증명서` / `경력증명서` / `기타증명서`
> - `{게시판명}`: 게시글이 등록된 게시판 이름 (예: `경영지원부서 공지`)

> **딥링크 및 발신 주체**
>
> | 이벤트 | 딥링크 | 발신 주체 |
> |--------|--------|---------|
> | 연차 신청 승인 | `leave-detail-done` (M-016) | 관리자 처리 |
> | 연차 신청 반려 | `leave-detail-reject` (M-017) | 관리자 처리 |
> | 증명서 발급 완료 | `cert-done` (M-034) | 관리자 처리 |
> | 증명서 발급 반려 | `cert-reject-detail` (M-035) | 관리자 처리 |
> | 새 공지사항 게시 | `board` (M-002) | 시스템 자동 |

#### 알림 수신함 (M-050)

수신된 모든 푸시 알림은 앱 내 알림 목록(M-050)에 영구 누적된다.

| 항목 | 규칙 |
|------|------|
| **진입** | 게시판 홈(M-002) 상단 알림 아이콘(🔔) 탭 |
| **미읽음 뱃지** | 읽지 않은 알림이 1건 이상이면 🔔 아이콘 우상단에 빨간 점(●) 표시. 전체 읽음 처리 후 자동 소멸 |
| **목록 구성** | 날짜별 섹션 헤더 + 알림 행(유형 아이콘·제목·내용·수신 시각). 미읽음 행은 연파랑 배경 + 우측 파란 점 |
| **수신 시각 표시** | 당일: "N분 전 / N시간 전" / 전날 이전: "YYYY.MM.DD HH:mm" |
| **탭 동작** | 알림 행 탭 → 읽음 처리 + 딥링크 이동. 대상 문서가 삭제된 경우 토스트 "해당 내역을 찾을 수 없습니다" |
| **모두 읽음** | 상단 [모두 읽음] 버튼 탭 → 전체 읽음 처리 + 뱃지 소멸 + 토스트 |

#### 알림 설정 (M-051)

| 설정 항목 | 기본값 | 동작 |
|---------|-------|------|
| 푸시 알림 받기 (전체) | ON | OFF 시 모든 알림 차단. 하위 항목 토글·방해금지 모드 비활성(dimmed) |
| 연차 / 휴가 | ON | 연차 신청 승인·반려 알림 개별 제어 |
| 증명서 | ON | 증명서 발급완료·반려 알림 개별 제어 |
| 영수증 / 결의서 | ON | 영수증 등록·결의서 관련 알림 개별 제어 |
| 게시판 / 공지 | ON | 새 공지사항 알림 개별 제어 |
| 방해금지 모드 | OFF | ON 시 시작·종료 시간 선택 UI 펼쳐짐. 매일 반복 적용. 설정 변경은 별도 저장 버튼 없이 즉시 반영(Auto-save) |
| 방해금지 시작 시간 | 추후 확정 (현재 미정) | 방해금지 모드 ON 시에만 노출 |
| 방해금지 종료 시간 | 추후 확정 (현재 미정) | 방해금지 모드 ON 시에만 노출 |

## 5. 용어 정의

| 용어 | 정의 |
|------|------|
| 직원 (Employee) | 모바일 앱을 주로 사용하는 일반 임직원. 사번 또는 이메일로 로그인. |
| 관리자 (Admin) | 인사담당자·경리담당자. PC에서 연차·증명서·FAQ를 관리. |
| 결재선 | 경리나라에서 자동 지정되는 결재 경로. 사용자가 직접 수정 불가. |
| 경리나라 | WEBCASH의 회계/경비 관리 SaaS. 영수증 매입 및 결재선 정보 제공. |
| 결의서 | 영수증 1건 이상을 묶어 경비 지출을 승인받는 전자 문서. |
| FAB | Floating Action Button. 화면 우하단 고정형 주요 액션 버튼. |
| SPA | Single Page Application. 페이지 전환 없이 JS 라우팅으로 화면을 교체. |
| 연차사용계획서 | 분기·월 단위로 예정 연차를 미리 신청하는 문서. |

## 10. Out of Scope — 이번 버전에 만들지 말 것

아래 기능은 이번 버전에 포함하지 않는다. 기획·개발·QA 단계에서 해당 항목은 구현 대상에서 제외한다.

| 제외 항목 | 사유 |
|---------|------|
| ❌ 급여 명세서 조회/발급 | 경리나라 연동으로 처리 — MY인사 자체 급여 기능 불포함 |
| ❌ 근태 관리 (출퇴근 기록) | 출퇴근 체크인/아웃, GPS 연동 등 근태 기능 전체 제외 |
| ❌ 조직도 / 인사발령 관리 | 직원 이동·승진·발령 등 인사 마스터 관리 제외 |
| ❌ 관리자 PC 게시판 관리 | 게시글 작성·삭제·관리는 모바일(직원)만 구현 |
| ❌ 경리나라 외 회계 시스템 연동 | 경리나라 이외 ERP/회계 솔루션 연동 불포함 |
| ❌ 결재선 수동 편집 | 결재선은 경리나라 기준 자동 지정 — 사용자 직접 변경 불가 |
| ❌ M-042 매뉴얼 (`manual`) | 이용 가이드 PDF 다운로드 화면. 더보기 메뉴에서 진입 경로 미제공 — 해당 화면 구현 제외. HTML 프로토타입에 화면 코드는 존재하나 고아 화면으로 실서비스에 미포함 |

## 11. Success Criteria — 완성 기준

### 11.1 모바일 — 직원용

- [ ] 사번/이메일로 로그인 후 사업장 선택 팝업에서 사업장 선택 → 게시판 화면 진입
- [ ] 아이디 찾기 / 비밀번호 변경 (휴대폰·이메일 인증)
- [ ] 사업장 전환 팝업에서 사업장 선택 후 전환
- [ ] 게시판 필터(≡)로 게시판별 필터링, 게시글 목록·상세 조회, 댓글 작성, 글 작성
- [ ] 연차 관리 화면에서 잔여 연차 및 결재 현황(진행중/승인/반려) 카드 확인
- [ ] 캘린더 날짜 드래그로 연차 신청 및 결재 상신
- [ ] 연차 취소 신청 (날짜·휴가 종류·취소 사유 선택) 및 연차사용계획서 제출
- [ ] 영수증 이미지 첨부 및 정보 입력 후 등록 (경리나라 자동 저장)
- [ ] 영수증 선택 후 결의서 작성 및 상신
- [ ] 재직/경력/기타 증명서 신청 (바텀 시트 모달) 및 처리 상태(진행중/발급완료/반려) 추적
- [ ] 발급완료 증명서 PDF 저장 또는 이메일 전송 (바텀 시트에서 수신 이메일 직접 입력)
- [ ] 인사 FAQ 열람 (단일 목록, 상세 조회)
- [ ] 더보기에서 내 정보(인사 마스터) 조회
- [ ] 게시판 홈 상단 알림(🔔) 아이콘으로 알림 목록(M-050) 진입. 미읽음 알림 있을 때 빨간 점 뱃지 표시
- [ ] 알림 목록에서 알림 행 탭 시 유형별 해당 화면으로 딥링크 이동 (연차 승인/반려, 증명서 발급완료/반려, 공지사항 5종 모두 검증)
- [ ] [모두 읽음] 탭 시 전체 읽음 처리 및 뱃지 소멸
- [ ] 더보기 → 알림 설정(M-051): 푸시 알림 전체 ON/OFF, 항목별 토글(연차·증명서·영수증·게시판) 동작
- [ ] 방해금지 모드 ON 후 시작·종료 시간 설정, 시간대 내 알림 미수신 확인

### 11.2 PC — 관리자용

- [ ] 대시보드 KPI 카드 확인 (대기 연차, 미답변 FAQ, 미처리 증명서, 총 직원수)
- [ ] 대시보드 미처리 업무에서 바로 FAQ 답변 처리
- [ ] 연차 관리: 직원명 검색·부서·연도 필터, 직원별 연차 현황 및 월별 사용 이력 조회, 총연차 더블클릭으로 수정, 결재 처리
- [ ] 업무신청관리: PDF 첨부 후 발급 완료 처리 및 반려 처리
- [ ] FAQ 관리: FAQ 등록·수정·삭제 및 노출 상태 관리
- [ ] 모바일 안내: 미설치 직원에게 카카오톡 앱 설치 안내 발송 (개별/일괄)

### 11.3 PC — 직원용

- [ ] 잔여 연차 및 사용 이력 조회
- [ ] 증명서 신청·이력 관리 (다운로드, 신청 취소, 반려 사유 확인)
- [ ] 프로필 개인 정보 수정 및 인사 정보(자격증·학력·어학) 항목 추가·수정·삭제
