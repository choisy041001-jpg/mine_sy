## 1. What — 한 줄 정의

MY인사는 연차 신청·결재, 증명서 발급, 영수증 기반 경비 결의서 작성 등 인사·경비 업무를 직원(모바일)과 인사담당자(PC)에게 제공하는 **중소기업용 HR 솔루션**입니다.

---

## 2. 메인 사용 시나리오

1. 직원(사용자)이 연차신청, 증명서 신청 등 인사업무 및 경비로 사용한 영수증 기반 결의서 작성, 공지글 확인 등을 할 때, **모바일 MY인사**를 사용한다.
2. 인사담당자(관리자)가 직원의 연차 사용현황 확인, 증명서 발급, 인사FAQ 관리 등 인사업무를 할 때, **PC MY인사**를 사용한다.

---

## 3. Who — 타겟 사용자

### 페르소나 1: 직원 (10인 미만 회사)

- **핵심 페인 포인트 1**: 외부에서 사용한 경비 영수증을 카톡이나 실물로 경리 직원에게 제출해야 하는 불편
- **핵심 페인 포인트 2**: 본인의 연차 현황을 파악하기 위해서는 인사담당자에게 별도 문의를 해야 하는 불편
- **핵심 페인 포인트 3**: 기관에 제출해야 하는 증명서 발급 요청을 카톡으로 하고, 진행현황을 확인하기 어려운 불편

### 페르소나 2: 인사담당자·경리담당자 (10인 미만 회사)

- **핵심 페인 포인트 1**: 직원이 카톡이나 전화로 연락하면 엑셀을 통해 수기로 연차 관리 및 확인해줘야 하는 불편
- **핵심 페인 포인트 2**: 직원이 외부에서 사용한 영수증을 카톡이나 실물로 전달받아 풀로 붙이고 관리해야 하는 불편
- **핵심 페인 포인트 3**: 증명서 발급 시 카톡으로 보내주다 보니 미처리된 업무 파악에 어려움

### 성공 정의

직원은 외부에서도 쉽고 편리하게 인사업무 신청 및 영수증 관리를 할 수 있으며, 인사담당자와 경리담당자는 정확하고 편리하게 업무를 처리 및 관리할 수 있음.

#### 화면 M-001: 로그인 (`login`)

- 증명서 관리 패널 (최근 신청 내역 목록, 증명서 신청 버튼)

**사용자 액션**:
- 증명서 신청 버튼 클릭 → 2단계 모달: Step 1 종류 선택(재직/경력/기타증명서) → Step 2 용도·메모 입력 → 신청하기

- 취소 버튼 → 변경 취소 후 조회 모드로 복귀
- 인사 정보 항목 수정 아이콘 → 수정 모달 (항목 편집 또는 삭제)

## 5. Errors — 예외 처리

| 화면 | 에러 케이스 | 처리 방식 |
|------|-----------|---------|
| M-001 로그인 | 인증 실패 | 에러 메시지 표시, 화면 유지 |
| M-009 아이디/비밀번호 | 인증번호 불일치 | 인증번호 불일치 안내 메시지 |
| M-011 연차 신청 | 날짜 미선택 | 날짜 선택 유도 메시지 |
| M-011 연차 신청 | 잔여 연차 부족 | "잔여 연차가 부족합니다" 인라인 알림 |
| M-014 연차 취소 | 취소할 휴가 종류 미선택 | 필드 에러 스타일 표시 |
| M-014 연차 취소 | 취소 사유 미선택 | 필드 에러 스타일 표시 |
| M-021 영수증 등록 | 필수 필드 미입력 (사용일자, 사용처, 금액) | 해당 필드 에러 스타일 + 안내 메시지 |
| M-023 결의서 | 영수증 0건 선택 | 결의서 작성 버튼 비활성화 |
| M-020 영수증 목록 | 이미 결의서 있는 영수증 선택 시도 | 체크박스 비활성화 (선택 불가) |
| M-031 증명서 신청 | 용도 미입력 | 필드 포커스 + 에러 스타일 표시 |
| M-031 증명서 신청 | 기타증명서 명칭 미입력 | 발급하기 버튼 비활성화 |
| M-031 증명서 신청 | 기타증명서 파일 미첨부 | 첨부 영역 에러 스타일, 버튼 비활성화 |
| P-003 발급처리 | PDF 미첨부 | 발급 완료 버튼 비활성화 |

## 8. Success Criteria — 완성 기준

### 모바일 (직원용)

- [ ] 사번/이메일로 로그인 후 게시판 화면 진입
- [ ] 아이디 찾기 / 비밀번호 변경 (휴대폰·이메일 인증)
- [ ] 사업장 전환 팝업에서 사업장 선택 후 전환
- [ ] 게시판 필터(≡)로 게시판별 필터링, 게시글 목록·상세 조회, 댓글 작성, 글 작성
- [ ] 연차 관리 화면에서 잔여 연차 및 결재 현황(진행중/승인/반려) 카드 확인
- [ ] 캘린더 날짜 드래그로 연차 신청 및 결재 상신
- [ ] 연차 취소 신청 (날짜·휴가 종류·취소 사유 선택) 및 연차사용계획서 제출
- [ ] 영수증 이미지 첨부 및 정보 입력 후 등록 (경리나라 자동 저장)
- [ ] 영수증 선택 후 결의서 작성 및 상신
- [ ] 재직/경력/기타 증명서 신청 (바텀 시트 모달) 및 처리 상태(진행중/발급완료/반려) 추적
- [ ] 발급완료 증명서 PDF 저장 또는 이메일 전송 (바텀 시트에서 수신 이메일 직접 입력)
- [ ] 인사 FAQ 열람 (카테고리별 목록·상세 조회)
- [ ] 더보기에서 내 정보(인사 마스터) 조회

### PC (관리자용)

- [ ] 대시보드 KPI 카드 확인 (대기 연차, 미답변 FAQ, 미처리 증명서, 총 직원수)

- [ ] FAQ 관리: FAQ 등록·수정·삭제 및 노출 상태 관리
