## 화면별 명세

### M-030 증명서 관리 (`cert-types`)

| 항목 | 내용 |
|------|------|
| **진입 조건** | 하단 탭바 '증명서' 탭 |
| **UI 요소** | 상단 타이틀 '증명서 관리' + 부제('증명서 발급 신청 및 이력 조회') / 증명서 종류 카드 3종(재직증명서·경력증명서·기타증명서) + 각 설명 / 발급 이력 목록(종류·용도·부수·신청일·상태 뱃지: 진행중/발급완료/반려) |
| **사용자 액션** | 종류 카드 탭 → `cert-form` (바텀 시트 모달) / 진행중 이력 탭 → `cert-status-detail` / 발급완료 이력 탭 → `cert-done` / 반려 이력 탭 → `cert-reject-detail` |
| **다음 화면** | `cert-form` (M-031), `cert-status-detail` (M-033), `cert-done` (M-034), `cert-reject-detail` (M-035) |

### M-031 증명서 신청 (`cert-form`)

| 항목 | 내용 |
|------|------|
| **진입 조건** | `cert-types`에서 종류 카드 탭 → `openCertModal()` 호출로 바텀 시트 모달 표시. 별도 화면 전환 없이 현재 화면 위에 오버레이로 표시됨 |
| **UI 요소** | 상단 핸들 바 / 종류명 소제목 + '발급 신청' 타이틀 / 닫기(×) 버튼 / 증명서 명칭 입력(기타증명서만) — **필수** / 용도 입력 — **필수** / 처리 안내 배너(예상 처리 시간·앱 알림 안내) / 발급하기 버튼 |
| **사용자 액션** | 발급하기 탭 → 용도 미입력 시 에러 표시·진행 차단 / 기타증명서 명칭 미입력 시 에러 표시·진행 차단 / 유효성 통과 시 모달 닫힘 후 `cert-status-detail`로 이동 / 닫기(×) 탭 → 모달 닫힘, `cert-types` 유지 |
| **다음 화면** | `cert-status-detail` (M-033) |
