## 데이터 표시 기준

**① 데이터 출처**
- 설치 현황은 **그룹웨어(경리나라) 사용자 목록 + MY인사 앱 로그인 기록**으로 산출한다. 앱 로그인 기록이 1회라도 있으면 **설치 완료**, 없으면 **미설치**다.
- 카카오 알림톡 발송 메시지는 외부 입력이 아닌 **시스템 고정 템플릿**이다(사용자 편집 불가).

**② 설치 현황 테이블**
- 열: 이름 / 부서 / 직급 / 연락처 / 설치 상태 / 최근 활동 / 카카오톡 발송. **미설치 사용자 우선 정렬**.
- **설치 상태**: `설치 완료`(녹색) / `미설치`(주황) 뱃지.
- **최근 활동**: 가장 최근 로그인 상대 시간(예: `5분 전`), 이력 없으면 `-`.
- **카카오톡 발송**: 미설치만 [카카오톡 발송] 버튼, 설치 완료는 `불필요` 표시.

**③ 카카오톡 발송 메시지**
- **고정 템플릿(읽기 전용)**: 사용자가 본문을 작성·편집할 수 없다. `{회사명}`은 **로그인한 회사명**으로, `{이름}`은 수신자명으로 시스템이 자동 치환한다(글자수 제한·카운터 없음). 화면에는 자리표시자가 아닌 치환된 실제 회사명이 표시된다.
- 미리보기는 치환 결과를 카카오톡 말풍선으로 표시(개별 = 실제 이름 치환 / 일괄 = `{이름}` 자리표시자 노출).

**④ 설치알림이력**
- 열: 발송일시(`YYYY.MM.DD HH:MM`) / 사용자 / 휴대폰번호 / 발송결과(`정상`·`오류`). 오류는 실패 사유 툴팁 표시.
