## 데이터 표시 기준

**① 데이터 출처**
- 신청 목록은 직원이 **모바일 증명서 신청(M-031)**에서 제출한 건이 관리자 업무신청관리로 수집된 것이다.
- **신청자·신청일·증명서 종류·제출처/용도**는 직원의 신청 데이터, **발급 PDF·인사담당자 메모·반려 사유**는 관리자(인사담당자)의 처리 데이터다.

**② 목록 테이블 표시 형식**
- **신청일**: `YYYY.MM.DD HH:MM`(24시간제), 신청일 기준 최신순 정렬(정렬 변경 UI 없음).
- **신청자**: 경리나라 직원관리 등록 직원명. 이 화면에서는 신청자 이름만 확인 가능하므로 이름으로만 검색하며, 사번은 목록에 표시하지도 검색 대상에 포함하지도 않는다.
- **증명서 종류**: `재직증명서` / `경력증명서` / `기타증명서` 색상 뱃지.
- **제출처/용도**: 직원이 신청 시 입력한 자유 텍스트.
- **상태**: `대기`(주황) / `발급완료`(녹색) / `반려`(빨강) 뱃지.
- **빈 상태**: 필터 결과가 없으면 "내역이 없습니다." 표시.

**③ 좌측 메뉴 숫자 뱃지**
- 좌측 사이드바 [업무신청관리] 메뉴의 숫자 뱃지는 처리 상태가 **'대기'인 건수**(대시보드 '미처리 증명서' KPI와 동일 기준)와 일치한다. **'대기' 건수가 0건이면 뱃지를 숨기지 않고 `0`으로 표시한다.** 연차 관리·FAQ 관리 메뉴에는 숫자 뱃지를 표시하지 않는다.

**④ 발급 이력(이력보기 모달) 표시 항목**
- 증명서 종류 / 신청자 / 신청 사유(제출처·용도) / 발급일시 / 발급 담당자 / 수령 방법(`MY인사 앱·PC(직원용) 다운로드` — 직원이 모바일 MY인사 앱 또는 PC(직원용)에서 발급 PDF 다운로드) / 첨부 PDF(파일명·용량) / 인사담당자 메모.
