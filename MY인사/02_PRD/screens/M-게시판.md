## 화면별 명세

### M-002 게시판 홈 (`board`)

| 항목 | 내용 |
|------|------|
| **진입 조건** | 로그인 성공 직후 기본 화면 또는 하단 탭바 '게시판' 탭 |
| **UI 요소** | 상단 타이틀 '게시판' + 부제('우리 회사 소식과 자유로운 대화') / 우상단 필터(≡) 아이콘 → 게시판 선택 바텀 시트 / 우상단 알림(🔔) 아이콘 → `notifications` 이동. 읽지 않은 알림이 1건 이상이면 아이콘 우상단에 빨간 점(●) 뱃지 표시, 전체 읽음 후 자동 소멸 / 최근 게시글 피드 목록(제목·NEW 뱃지·게시판명·작성자·날짜) / 글쓰기 FAB(+) |
| **사용자 액션** | 필터(≡) 아이콘 탭 → 게시판 선택 바텀 시트(전체 + 사업장 운영 게시판 동적 로드; *예시: 자유게시판 / 경조사게시판 등*) / 알림(🔔) 아이콘 탭 → `notifications` (M-050) / 게시글 행 탭 → `post-list`(해당 게시판 목록) / FAB 탭 → `compose` |
| **다음 화면** | `post-list` (M-003), `compose` (M-005), `notifications` (M-050) |
