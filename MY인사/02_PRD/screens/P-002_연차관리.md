## 데이터 연동

**① 전자결재 완료 시 자동 반영**
연차 사용·취소에 대한 전자결재가 경리나라 결재함에서 완료되면, 연차 관리의 해당 직원 연차(사용 일수·잔여 일수)가 자동으로 갱신된다. 관리자가 수동 입력 없이 결재 완료 즉시 반영된다.

**② 잔여 연차 초기 불러오기**
직원을 연차 관리에 처음 등록할 때 잔여 연차를 외부 시스템에서 자동으로 가져온다.
- 경리나라 **인사급여** 이용 고객사: 인사급여의 잔여 연차를 불러온다.
- 인사급여 미사용이나 결재함에서 잔여 연차 확인 가능한 경우: 결재함의 잔여 연차를 불러온다.
- 두 시스템 모두 미사용 고객사: 총 연차 셀 더블클릭 수동 편집 방식을 유지한다.

**③ 인사급여 상시 동기화**
경리나라 인사급여 이용 고객사는 직원별 연차 데이터가 인사급여와 자동 동기화된다. 일반근로자 불러오기 기능은 경리나라 직원관리의 일반근로자 목록을 호출하여 직원을 일괄 등록한다.

**④ 연도 선택 기본값 및 범위 규칙**
- 화면 최초 진입 시 연도 필터는 오늘 날짜 기준 현재 연도(yyyy)로 자동 설정된다.
- 사용자가 연도를 변경하면 해당 연도의 연차 현황 데이터로 즉시 갱신된다. 연도 간 데이터는 독립적으로 관리된다.

**⑤ 일반근로자 불러오기 연도 귀속**
일반근로자 불러오기로 등록된 직원 데이터는 **실행 시점에 선택된 연도에만 귀속**된다. 다른 연도의 데이터에는 영향을 주지 않는다.

**⑥ 엑셀 다운로드 범위**
엑셀 다운로드는 현재 선택된 연도 및 적용 중인 필터(부서·직원명) 기준으로 화면에 표시된 목록과 동일한 데이터를 내보낸다. 포함 컬럼: 직원명 / 부서 / 총 연차 / 1월~12월 월별 사용 현황 / 잔여 연차.
