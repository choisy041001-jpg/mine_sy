## 화면별 명세

### P-001 관리자 대시보드 (`dashboard`)

| 항목 | 내용 |
|------|------|
| **진입 조건** | 관리자 계정 로그인 직후 기본 화면, 또는 사이드바 '대시보드' 클릭 |
| **UI 요소** | KPI 카드 4개(승인할 연차 / 미처리 증명서 / 노출중인 질문 / 총 직원수) / 미처리 업무 패널: 대기중 증명서 행(발급 버튼) + 대기중 연차 신청 행(승인 버튼) — 요청 일시는 `YYYY-MM-DD HH:MM` 절대 일시로 표기 / 직원 현황 패널: 전체/재직/휴직/퇴직 탭 + 직원 테이블(이름·부서·직급·입사일·연락처·상태) |
| **KPI 데이터 기준** | **승인할 연차**: 로그인한 관리자 본인이 결재해야 하는 연차 결재 대기 건수(① 연차 사용 신청(휴가신청서) ② 연차 취소 신청). 경리나라 결재함 기준 본인이 현재 결재 차례인 건만 집계, 처리 완료·타인 결재 건 제외. <br>**미처리 증명서**: 업무신청관리(`cert`) 처리 상태 '대기' 건수(발급완료·반려 제외). <br>**노출중인 질문**: FAQ 관리(`faq`) 노출 상태 '노출 중' 건수(비노출 제외). <br>**총 직원수**: 경리나라 직원관리 등록 일반근로자 전체 인원(재직·휴직·퇴직 포함). <br>모든 수치는 페이지 로드 시 1회 갱신, 실시간 폴링 없음. |
| **사용자 액션** | KPI 카드 클릭 → 해당 관리 페이지(`leave`·`cert`·`faq`, '총 직원수'는 클릭 불가) / 미처리 업무 증명서 행 또는 발급 버튼 → `cert` 페이지 이동 / **미처리 업무 연차 신청 행의 '승인' 버튼 클릭 → 경리나라 결재함 앱을 새 창(별도 탭)으로 오픈** — 실제 결재 처리는 결재함 앱에서 수행 |
| **다음 화면** | `leave` (P-002), `cert` (P-003), `faq` (P-004), **결재함 앱 (새 창)** |
