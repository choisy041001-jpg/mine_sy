## 데이터 표시 기준

**① 데이터 출처**
- 연차 데이터는 **경리나라 결재함 집계 결과**(P-002와 동일 기준 — 최종결재/접수 완료 건만 반영)에서 **로그인한 직원 본인 1인**의 값을 가져온다. 증명서 데이터는 **직원 본인의 증명서 신청 이력**이다.
- 본 화면은 **조회 전용**이며, 연차 신청·취소는 모바일 MY인사 앱에서만 처리한다.

**② 잔여 연차 카드**
- 잔여 일수(대형) = `총 연차 − 사용`. `총 {N}` / `사용 {N}` 칩 표시. 진행 막대 채움 = `사용 ÷ 총`(%).
- 본인 계정·현재 연도(yyyy) 기준. 최종결재(접수) 완료 건만 집계.

**③ 앱 설치 QR 카드**
- 설치 URL `my-insa.app/install` *(예시 URL — 실제 서비스 도메인 확정 후 교체)* 를 인코딩한 QR + URL 칩. 설치 여부와 무관하게 상시 표시(안내 전용, 액션 버튼 없음).

**④ 연차 사용 이력 테이블**
- 본인의 현재 연도 이력만, 사용일자 최신순. 열: 날짜(`YYYY.MM.DD (요일)`) / 구분(연차·오전 반차·오후 반차) / 일수(`N일`) / 사유 / 상태 뱃지(`대기`·`승인`).
- 빈 상태: **"내역이 없습니다."**

**⑤ 증명서 관리 패널**
- 최근 신청 내역 최대 5건. 각 행: 증명서 종류 / `{신청일} · {용도}` / 상태 뱃지(`대기`·`발급완료`·`반려`).
- [증명서 신청] 버튼 클릭 시 P-011로 이동. 빈 상태: **"내역이 없습니다."**
