## 데이터 표시 기준

**① 데이터 출처**
- 모든 데이터는 **직원 본인 계정** 기준이다. 다른 직원의 프로필은 조회·수정할 수 없다.
- 인사 기본 정보(사번·입사일·부서·직급·고용형태)와 프로필 사진 하단 이름·부서·직급은 **경리나라 직원관리** 출처이며 읽기 전용이다.

**② 프로필 사진 + 인사 기본 정보(좌측)**
- 프로필 사진: 등록 이미지, 미등록 시 이름 첫 글자 이니셜 아바타(88×88 px). 카메라 아이콘은 편집 모드에서만 노출.
- 인사 기본 정보: 사번 / 입사일(`YYYY.MM.DD`) / 부서 / 직급 / 고용형태 — 읽기 전용(편집 모드에서도 비활성).

**③ 개인 정보(우측, 수정 가능)**
- 조회 모드: 이름·영문 이름·이메일·연락처 2×2 그리드. 편집 모드: 입력 필드 전환.
- 형식·제한: 이름 30자(필수) / 영문 이름 50자·영문·공백·하이픈(선택) / 이메일 100자·형식 검증(필수, 로그인 아이디와 무관) / 연락처 20자·숫자만·하이픈 자동(필수).
- 저장 시 경리나라 사용자관리·직원관리와 즉시 동기화.

**④ 내 인사 정보(우측, 수정 가능)**
- 유형: 자격증 / 학력 / 어학 / 수상·표창 / 기타 (등록된 유형 섹션만 노출). 항목당 최대 100자, 다중 추가 가능.
- 카드: 유형별 색상 아이콘 + 명칭 + 보조 정보(`{기관} · {취득일/기간}`). [항목 추가]·수정 아이콘은 편집 모드에서만 노출.

**⑤ 인사 정보 추가/수정 팝업**
- 추가: 항목 유형(필수) 선택 → 유형별 동적 라벨(명칭 필수 / 기관 / 취득일·기간 / 점수·등급 / 메모). 저장 토스트 `✅ 인사 정보가 추가되었습니다.`.
- 수정: 유형별 라벨 4필드(명칭 필수) + [삭제]/[취소]/[저장]. 저장 토스트 `✅ 인사 정보가 수정되었습니다.`, 삭제 토스트 `항목이 삭제되었습니다.`.

**⑥ 모드 전환·사진 변경**
- [수정]→편집 모드, [저장]→토스트 `✅ 프로필이 저장되었습니다.` 후 조회 복귀, [취소]→원복.
- 프로필 사진: PNG·JPG / 최대 50 MB / 88×88 px 권장. 파일 선택 시 토스트 `✅ 프로필이 설정되었습니다.`.
- 편집 중 다른 메뉴 이탈 시 경고 없이 초기화.
