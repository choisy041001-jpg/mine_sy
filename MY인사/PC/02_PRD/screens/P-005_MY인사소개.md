## 데이터 표시 기준

**① 데이터 출처**
- 본 화면은 **정적 마케팅 콘텐츠**(하드코딩 카피·목업)로만 구성된다. 외부 연동·실시간 집계·사용자별 분기가 없으며 직원·관리자 공통으로 동일하게 노출된다.
- 모바일 목업은 **실제 MY인사 기능 범위와 일치**해야 한다. 미구현·범위 외 기능을 광고하지 않는다.

**② 기능 소개 블록(4종)**
- 좌우 교차 배치. 각 블록 = 번호 / eyebrow / 타이틀 / 설명 / 3개 리스트 / 모바일 목업.
- **02 증명서 블록**: 카피·목업은 **실제 수령 방식과 일치**해야 한다 — 발급 완료 후 **MY인사 앱·PC에서 PDF 직접 다운로드**(이메일 발송 아님).
- **04 FAQ 블록**: 목업은 **읽기 전용 FAQ 목록**(질문 카드 + 답변 요약). MY인사 FAQ는 **카테고리 구분·1:1 질문/답변 기능이 없는 단일 읽기 목록**이므로 목업에 '내 질문'/'자주 묻는 질문' 탭, 카테고리 뱃지, 답변 대기/완료 상태, 질문 등록 FAB를 표시하지 않는다.

**③ 가격 정책**
- 본 화면에서는 **가격 정책을 안내하지 않는다**(가격 카드 섹션 미노출).
