## 데이터 표시 기준

**① 데이터 출처**
- FAQ는 외부 연동 없이 **관리자가 P-004에서 직접 등록·수정·삭제**하는 데이터다. 카테고리 구분은 없으며(단일 목록), 직원 1:1 질문에 답변하는 기능은 없다.
- **'노출 중'** 상태 FAQ만 직원 모바일 앱(M-006/M-008) FAQ에 그대로 노출되고, **'비노출'** FAQ는 직원에게 표시되지 않는다.

**② 목록 테이블 표시 형식**
- 열 구성: **질문 / 답변 요약 / 노출 상태 / [수정]**. **등록 순서 기준 정렬**(정렬 변경 UI 없음).
- **질문**: 관리자가 등록한 질문 텍스트. 단일 라인, 길면 말줄임(`…`).
- **답변 요약**: 답변 본문 앞부분 요약 + 말줄임(`…`), 보조 색상. 전체 답변은 수정 모달에서 확인.
- **노출 상태**: `노출 중`(녹색) / `비노출`(회색) **읽기 전용 뱃지**. 전환은 수정 모달의 노출 상태 select로만 처리(목록 직접 토글 없음).
- **빈 상태**: 등록된 FAQ가 없으면 "내역이 없습니다." 표시.

**③ 등록·수정 입력 항목**
- **노출 상태**(select): `노출 중`(기본) / `비노출`.
- **질문**(필수): 단일 라인, 최대 200자.
- **답변**(필수): 다중 라인, 최대 2,000자, 카운터 표시. 입력 내용은 직원 앱 FAQ에 그대로 노출.

**④ 노출 상태 집계**
- 대시보드 '노출중인 질문' KPI 카드는 노출 상태가 **'노출 중'** 인 FAQ 건수를 집계한다. '비노출' 건은 제외된다.

**⑤ 좌측 메뉴 숫자 뱃지**
- 좌측 사이드바 [FAQ 관리] 메뉴에는 숫자 뱃지를 표시하지 않는다.
