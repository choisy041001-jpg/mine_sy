## 세부 설계 요건 — 데이터 표시 기준

> 본 화면의 모든 데이터는 **직원 본인 계정** 기준이다. 다른 직원의 프로필은 조회·수정할 수 없다.

### 1. 좌측 — 프로필 사진 + 인사 기본 정보

- **프로필 사진 패널**
  - 사진이 등록된 경우 등록 이미지를, 미등록 시 **이름 첫 글자 이니셜 아바타**(88×88 px, `border-radius:22px`, accent 그라디언트 배경)를 표시한다.
  - 아바타 하단에 **이름**(굵게) / **부서 · 직급**(`{부서} · {직급}`)을 표시한다. 출처는 경리나라 직원관리.
  - 사진 변경용 **카메라 아이콘**(아바타 우상단)은 **편집 모드에서만** 노출되며, 클릭 시 파일 선택 창(`image/*`)을 연다.
- **인사 기본 정보 패널**(읽기 전용)
  - 행: 사번 / 입사일(`YYYY.MM.DD`) / 부서 / 직급 / 고용형태. 좌측 라벨(회색) + 우측 값(굵게)의 행 구조.
  - 출처는 **경리나라 직원관리**이며 직원 본인은 수정할 수 없다(편집 모드에서도 비활성).

### 2. 우측 — 개인 정보 패널

- **조회 모드**: 이름 / 영문 이름 / 이메일 / 연락처를 **2×2 그리드**(라벨 + 값)로 표시한다.
- **편집 모드**: 동일 4개 항목을 입력 필드로 전환한다. 필수 항목(이름·이메일·연락처)은 라벨 옆 빨간 `*`로 표시하며, **영문 이름은 선택**이다.
- **항목별 입력 형식·제한**
  - 이름: 최대 30자(필수).
  - 영문 이름: 최대 50자, **영문·공백·하이픈만 허용**(선택).
  - 이메일: 최대 100자, 이메일 형식 검증(필수). **로그인 아이디와 무관**하며 그룹웨어 로그인에 영향을 주지 않는다.
  - 연락처: 최대 20자, **숫자만 입력 → 하이픈 자동 삽입**(필수).
- **저장 동기화**: 이름·연락처 등 개인 정보 저장 시 **경리나라 사용자관리·직원관리와 즉시 동기화**된다. 이메일은 동기화 대상이나 로그인 아이디와는 무관.

### 3. 우측 — 내 인사 정보 패널

- 항목을 **유형 섹션 단위**로 묶어 표시한다. 프로토타입 샘플 섹션은 자격증·학력·어학이며, 추가 모달에서 **수상·표창·기타**까지 총 5종을 추가할 수 있다(등록된 유형의 섹션만 노출).
- 각 항목은 **카드** 형태로, 좌측 **유형별 색상 아이콘 박스**(자격증 `#dbeafe`/`#1e40af`, 학력 `#f3e8ff`/`#7c3aed`, 어학 `#fef9c3`/`#92400e`) + **명칭**(굵게) + 보조 정보(`{기관} · {취득일/기간}`)를 표시한다.
- **[항목 추가]** 버튼(패널 헤더)과 각 카드의 **수정 아이콘**은 **편집 모드에서만** 노출된다.
- 다중 항목 추가 가능(추가 수 제한 없음), 항목당 최대 100자.

### 4. 인사 정보 추가 팝업 (`항목 추가`)

- **항목 유형**(필수, select): 자격증 / 학력 / 어학 / 수상·표창 / 기타. 선택 시 하위 입력 필드가 노출되며 **유형별로 라벨·placeholder가 동적 변경**된다.
  - 자격증 — 자격증명(필수) / 발급 기관 / 취득일 / 등급·종류(선택).
  - 학력 — 학교명·전공(필수) / 학위 / 재학 기간 / 졸업 구분(선택).
  - 어학 — 시험명(필수) / 주관 기관 / 취득일 / 점수·등급(선택).
  - 수상·표창 — 수상명(필수) / 수여 기관 / 수상일 / 등급(선택).
  - 기타 — 항목명(필수) / 기관·출처 / 날짜·기간 / 세부 내용(선택).
- 공통으로 **메모**(선택, 멀티 라인) 입력란을 둔다.
- [저장] 시 유형·명칭(필수값) 검증 후 팝업을 닫고 토스트 `✅ 인사 정보가 추가되었습니다.`를 표시한다.

### 5. 인사 정보 수정 팝업 (`항목 수정`)

- 카드의 수정 아이콘 클릭 시 해당 항목 값을 채워 표시한다. 유형별 라벨 4개 필드(명칭은 필수) + [삭제] / [취소] / [저장] 버튼.
- [저장] 시 필수값 검증 후 토스트 `✅ 인사 정보가 수정되었습니다.`, [삭제] 시 토스트 `항목이 삭제되었습니다.`를 표시한다.

### 6. 모드 전환 · 프로필 사진 변경

- **[수정]** → 편집 모드(개인 정보 입력 활성화 + 사진 카메라 아이콘 + 내 인사 정보 [항목 추가]·수정 아이콘 노출). **[저장]** → 토스트 `✅ 프로필이 저장되었습니다.` 후 조회 모드 복귀. **[취소]** → 변경 내용 초기화 후 조회 모드 복귀.
- **프로필 사진 변경**: PNG·JPG 지원, 최대 50 MB, **88×88 px 권장**(초과·미달 이미지는 화면 맞춤 자동 조정). 파일 선택 시 토스트 `✅ 프로필이 설정되었습니다.`.
- **편집 중 메뉴 이탈**: 편집 모드에서 다른 사이드바 메뉴로 이동 시 별도 경고 없이 변경 내용이 저장되지 않고 초기화된다.
